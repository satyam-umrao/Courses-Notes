Python : Short_Notes [Day-1]

Introduction of Python

High-level language vs low-level language:

High-level programming languages are programmer-friendly, easily understandable, and easier to debug. Examples: Java, C, C++, Python. Low-level languages are machine-friendly and complex to debug. Examples: Assembly, Machine language.

Compiler vs Interpreter

A compiler converts source code to machine code at once. Examples: Java, C++. An interpreter runs code line by line. Examples: PHP, Python.

Scripting Language

Programming languages that are interpreted. Used for automation and run line by line.

Definition of Python

Python is a general-purpose, interpreted, high-level programming language. It is easy to understand, debug, and open-source. Interpreted as well as compiled. First compiled to bytecode, then interpreted.

History of Python

Invented by Guido Van Rossum in the Netherlands in the early '90s. Released in 1991. Derived from ABC language.

Why the name Python?

Named after "Monty Python’s Flying Circus" - a popular TV show Guido enjoyed.

Python Versions

  • Python 1.0 - Jan 1994
  • Python 2.0 - Oct 2000
  • Python 3.0 - Dec 2008

Note: Python 3 is not backward-compatible with Python 2.
